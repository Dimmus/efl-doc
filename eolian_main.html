Introduction

The Eolian EO file parser and code generator.

How to compile

Eolian is a library your application links to. The procedure for this is very simple. You simply have to compile your application with the appropriate compiler flags that the pkg-config script outputs. For example:

Compiling C or C++ files into object files:

gcc -c -o main.o main.c `pkg-config --cflags eolian`

Linking object files into a binary executable:

gcc -o my_application main.o `pkg-config --libs eolian`

See pkgconfig
